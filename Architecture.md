## ğŸ›¡ï¸ Security Considerations

### Forward Secrecy
- New message keys for each message
- Compromised key cannot decrypt past messages
- DH ratchet provides long-term forward secrecy

### Break-in Recovery
- New DH ratchet step generates fresh keys
- Attacker with temporary access loses access after ratchet

### Replay Protection
- Incrementing nonces
- Message deduplication
- Timestamp validation

### Authentication
- GCM authentication tag
- Sender public key verification
- Safety numbers for contact verification
