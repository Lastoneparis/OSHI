# OSHI Messenger - Protocol Documentation

## Overview

OSHI implements a secure messaging protocol inspired by the Signal Protocol, with additions for mesh networking and decentralized message delivery.

## ğŸ”„ Double Ratchet Protocol

### What is Double Ratchet?

The Double Ratchet algorithm combines two cryptographic ratchets:

1. **Diffie-Hellman Ratchet** - Provides break-in recovery
2. **Symmetric Key Ratchet** - Provides forward secrecy per message
